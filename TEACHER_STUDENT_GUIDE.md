# Teacher and Student User Guide - INR99 Offline School Server

## 👩‍🏫 Teacher User Guide

### Getting Started with INR99 Offline School Server

#### **First Login**
1. Open your browser and go to `http://your-school-server:8080`
2. Click on "Sign In" 
3. Enter your email and password provided by your school administrator
4. Select "Teacher" as your role
5. Click "Sign In" to access your INR99 dashboard

#### **Dashboard Overview**
Your teacher dashboard includes:
- **My Classes**: View and manage your assigned classes
- **Student Progress**: Monitor individual and class performance
- **Lesson Plans**: Create and organize your teaching materials
- **Assignments**: Create, distribute, and grade student work
- **AI Assistant**: Access teaching support and resources
- **Digital Library**: Browse educational content and resources

### Daily Teaching Workflow

#### **Morning Preparation (15 minutes)**
```
1. Log in and check dashboard
2. Review student homework completion
3. Check for school announcements
4. Preview today's lesson content
5. Prepare any additional materials
```

#### **During Class Teaching**
```
1. Start with digital attendance
2. Introduce lesson using multimedia content
3. Use interactive elements for engagement
4. Assign in-class digital activities
5. Monitor student progress in real-time
6. Use AI assistant for difficult concepts
7. Assign homework before class ends
```

#### **After Class Follow-up (15 minutes)**
```
1. Review class participation and engagement
2. Grade submitted assignments
3. Update student progress records
4. Identify students needing extra help
5. Plan tomorrow's lessons
6. Communicate with parents if needed
```

### Key Features and How to Use Them

#### **📚 Digital Library**
**What it is**: Access to thousands of educational resources including Wikipedia, textbooks, videos, and interactive content.

**How to use it**:
1. Click "Digital Library" from your dashboard
2. Use search bar to find content by subject, topic, or keyword
3. Filter by grade level, content type, or subject
4. Preview content before sharing with students
5. Add frequently used resources to "My Favorites"

**Best Practices**:
- Preview all content before showing students
- Bookmark resources for future lessons
- Use advanced search for specific topics
- Check content age-appropriateness
- Download content for offline use when needed

#### **🤖 AI Teaching Assistant**
**What it is**: Your personal AI assistant for lesson planning, student support, and teaching strategies.

**How to use it**:
1. Click "AI Assistant" from any page
2. Ask questions in natural language
3. Use for:
   - Lesson plan ideas
   - Concept explanations
   - Assignment creation
   - Student engagement strategies
   - Differentiated instruction

**Example Prompts**:
```
• "Create a lesson plan for photosynthesis for Grade 7"
• "Explain gravity in simple terms for 10-year-olds"
• "Suggest ways to help struggling math students"
• "Give me ideas for interactive history lessons"
• "How can I teach fractions using real-world examples?"
```

#### **📊 Student Progress Tracking**
**What it is**: Real-time monitoring of student performance, engagement, and learning outcomes.

**How to use it**:
1. Click "Student Progress" from dashboard
2. View class overview with completion rates
3. Click on individual students for detailed progress
4. Filter by subject, date range, or activity type
5. Export reports for parent meetings

**Key Metrics to Monitor**:
- Assignment completion rates
- Quiz and test scores
- Time spent on learning activities
- AI assistant usage patterns
- Areas where students struggle

#### **📝 Assignment Creation**
**What it is**: Tools for creating, distributing, and grading digital assignments.

**How to create assignments**:
1. Click "Create Assignment" from dashboard
2. Choose assignment type:
   - Quiz: Multiple choice, true/false, short answer
   - Essay: Written responses with word counts
   - Project: Multi-step assignments with uploads
   - Interactive: Drag-and-drop, matching exercises
3. Set due date and time
4. Add instructions and resources
5. Assign to specific classes or students
6. Save and publish

**Grading Features**:
- Automatic grading for quizzes
- Rubric-based grading for essays
- Audio and video feedback
- Bulk grading options
- Grade book integration

#### **👥 Class Management**
**What it is**: Tools for managing student accounts, attendance, and communication.

**Features available**:
- Student roster management
- Attendance tracking
- Class announcements
- Parent communication
- Group project creation
- Behavior tracking

### Troubleshooting Common Issues

#### **Technical Issues**
```
Problem: Students can't log in
Solution: 
• Check student credentials
• Verify account is active
• Clear browser cache
• Try different browser

Problem: Content won't load
Solution:
• Check internet connection to server
• Refresh the page
• Try different device
• Contact IT support

Problem: AI assistant not responding
Solution:
• Check if service is running
• Try refreshing the page
• Use simpler prompts
• Report issue to administrator
```

#### **Teaching Challenges**
```
Problem: Students not engaging with digital content
Solution:
• Make content more interactive
• Use gamification elements
• Connect to student interests
• Provide choices in activities

Problem: Students cheating with AI
Solution:
• Design assignments that require original thinking
• Use in-class assessments
• Teach ethical AI usage
• Focus on process, not just answers

Problem: Managing different skill levels
Solution:
• Use flexible grouping
• Provide tiered assignments
• Offer choice in difficulty
• Use AI for personalized support
```

## 🎒 Student User Guide

### Getting Started

#### **First Login**
1. Open your browser and go to `http://your-school-server:8080`
2. Click "Sign In"
3. Enter your student ID and password given by your teacher
4. Select "Student" as your role
5. Click "Sign In" to start learning

#### **Your Dashboard**
Your student dashboard shows:
- **My Progress**: How you're doing in each subject
- **Today's Lessons**: What you need to learn today
- **My Assignments**: Homework and classwork to complete
- **AI Assistant**: Your 24/7 homework helper
- **Digital Library**: All your learning materials
- **Achievements**: Badges and rewards you've earned

### Daily Learning Routine

#### **Before School (10 minutes)**
```
1. Log in to your dashboard
2. Check today's schedule and lessons
3. Review feedback on yesterday's work
4. Note questions for your teacher
5. Preview today's topics
```

#### **During Class**
```
1. Follow your teacher's digital lesson
2. Participate in interactive activities
3. Take notes on important concepts
4. Ask AI assistant when you're confused
5. Complete in-class exercises
6. Save questions for after class if needed
```

#### **After School Study (30-60 minutes)**
```
1. Complete today's homework
2. Use AI assistant for difficult problems
3. Review lessons you didn't understand
4. Study for upcoming tests
5. Explore topics that interest you
6. Track your progress
```

### Using Key Features

#### **📚 Digital Library**
**Your personal library with thousands of learning resources**

**How to find what you need**:
1. Click "Digital Library"
2. Type what you're looking for in the search bar
3. Use filters for subject, grade level, or type
4. Click on items to open and read/watch
5. Add favorites to find them again easily

**What you can find**:
- Wikipedia articles made for students
- Textbooks and reference materials
- Educational videos and animations
- Interactive exercises and games
- Stories and reading materials

**Tips for using the library**:
- Use specific keywords for better results
- Try different search terms if you don't find what you need
- Save useful resources in "My Favorites"
- Check the grade level to make sure content is right for you
- Use the table of contents to navigate long articles

#### **🤖 AI Teaching Assistant**
**Your smart homework helper available 24/7**

**How to get help**:
1. Click "AI Assistant" from any page
2. Type your question in simple language
3. The AI will help you step-by-step
4. Ask follow-up questions if you need more help

**What you can ask for help with**:
```
Homework Problems:
• "Can you help me solve this math problem: 2x + 5 = 15?"
• "Explain photosynthesis in simple terms"
• "Help me understand this history question about..."

Concept Explanations:
• "What is gravity and how does it work?"
• "Why do we need to learn fractions?"
• "Can you explain the water cycle?"

Learning Strategies:
• "How can I study better for tests?"
• "What's the best way to memorize vocabulary?"
• "Help me make a study plan for my exams"

Writing Help:
• "Can you help me start this essay about pollution?"
• "Is this paragraph grammatically correct?"
• "Help me think of ideas for my story"
```

**Tips for using AI Assistant**:
- Be specific in your questions
- Ask one question at a time
- Tell the AI what you already understand
- Use it for learning, not just getting answers
- Always double-check important information

#### **📊 My Progress**
**Track how well you're learning and improving**

**What you can see**:
- Your completion rates for each subject
- Quiz and test scores
- Time spent on different activities
- Strengths and areas to improve
- Learning streaks and achievements

**How to use progress tracking**:
1. Click "My Progress" from dashboard
2. Look at your overall progress first
3. Click on subjects to see detailed progress
4. Set goals for improvement
5. Celebrate your achievements!

**Making Progress Goals**:
```
Weekly Goals:
• Complete 90% of my assignments
• Spend 30 minutes each day on math
• Ask AI for help when I'm stuck
• Review one difficult concept each week

Monthly Goals:
• Improve my science grade by 10%
• Read 5 new books from the library
• Master 3 new math skills
• Help 3 classmates with their work
```

#### **📝 Assignments**
**Complete and submit your school work digitally**

**Types of assignments**:
- **Quizzes**: Multiple choice and short answers
- **Essays**: Writing assignments with specific requirements
- **Projects**: Multi-step assignments with different parts
- **Interactive**: Drag-and-drop and matching activities

**How to complete assignments**:
1. Click "My Assignments" from dashboard
2. Look for assignments with "Due Soon" status
3. Click on an assignment to see details
4. Complete the work online or upload files
5. Review your work before submitting
6. Click "Submit" when you're finished

**Tips for success**:
- Start assignments early, not at the last minute
- Read instructions carefully before starting
- Use AI Assistant for help, but do your own work
- Save your work often
- Check your work for mistakes before submitting

#### **🏆 Achievements and Rewards**
**Earn badges and recognition for your learning**

**Types of achievements**:
- **Learning Streaks**: Consecutive days of learning
- **Subject Mastery**: Excelling in specific subjects
- **Helping Others**: Assisting classmates
- **Creative Thinking**: Original ideas and solutions
- **Persistence**: Not giving up on difficult problems

**How to earn achievements**:
- Log in and learn every day
- Complete assignments on time
- Help your classmates
- Ask thoughtful questions
- Try your best on challenging work

### Learning Strategies

#### **Effective Study Habits**
```
Daily Learning (30-60 minutes):
• Review what you learned in class
• Complete your homework
• Use AI Assistant for difficult parts
• Read ahead for tomorrow's lessons
• Track your progress

Weekly Review (2-3 hours):
• Go over all subjects from the week
• Identify topics you need to review
• Practice problems in weak areas
• Set goals for next week
• Celebrate your progress

Test Preparation:
• Start studying 3-4 days before tests
• Use AI Assistant to explain difficult concepts
• Practice with sample questions
• Get enough sleep before test day
• Stay calm and confident during tests
```

#### **Using AI Assistant Effectively**
```
Good Questions:
• "Can you explain step-by-step how to solve...?"
• "I'm stuck on this problem. Can you help me understand?"
• "What's the difference between these two concepts?"
• "Can you give me an example of...?"
• "How does this relate to what I learned before?"

Not So Good:
• Just asking for answers without trying
• Copying AI responses without understanding
• Using AI for tests or quizzes
• Asking the AI to do your work for you

Best Practices:
• Always try to solve problems yourself first
• Use AI to understand, not to get answers
• Ask follow-up questions to learn more
• Thank the AI and be polite
• Use what you learn in your own words
```

#### **Collaborating with Classmates**
```
How to Work Together:
• Share helpful resources you find
• Explain concepts to each other
• Study together for tests
• Give feedback on each other's work
• Celebrate each other's successes

Digital Citizenship:
• Be respectful in all communications
• Help others who are struggling
• Share your own knowledge
• Give credit when using others' ideas
• Report problems to teachers
```

### Staying Safe and Responsible

#### **Online Safety Rules**
```
Always Remember:
• Keep your password private
• Log out when you're finished
• Tell a teacher about problems
• Be kind to others online
• Only use school-approved websites

Never Share:
• Your password with anyone
• Personal information online
• Pictures of yourself or others
• Mean or hurtful messages
• Inappropriate content
```

#### **Academic Honesty**
```
Using AI Assistant the Right Way:
✓ Use it to understand difficult concepts
✓ Get help when you're stuck
✓ Ask for explanations and examples
✓ Use it for studying and review

Using AI Assistant the Wrong Way:
✗ Asking it to do your homework
✗ Copying answers without understanding
✗ Using it during tests or quizzes
✗ Pretending AI work is your own
✗ Depending on it for all your work
```

#### **Taking Care of Equipment**
```
Device Care:
• Keep food and drinks away from devices
• Use devices on flat, stable surfaces
• Carry devices carefully with two hands
• Report any problems immediately
• Follow all school technology rules

Digital Responsibility:
• Use school devices for learning only
• Respect school bandwidth limits
• Download only approved content
• Keep your workspace organized
• Log out completely when finished
```
