## 📖 What is INR99 Offline School Server?

INR99 Offline School Server is a comprehensive **digital education platform** designed specifically for schools in areas with limited or unreliable internet connectivity. It brings world-class educational resources, AI-powered teaching assistants, and interactive learning tools to your students — **all working completely offline**.

## 🚀 Quick Start Guide

### Installation (30 Minutes)
```bash
# One-command installation for Debian-based systems
sudo apt-get update && sudo apt-get install -y curl && \
curl -fsSL https://raw.githubusercontent.com/Crosstalk-Solutions/project-nomad/refs/heads/main/install/install_nomad.sh -o install_inr99_school_server.sh && \
sudo bash install_inr99_school_server.sh
```

### Access Your Server
1. Open browser to `http://localhost:8080` or `http://YOUR_SERVER_IP:8080`
2. Run "School Setup" wizard (5 minutes)
3. Download content collections for your school level
4. Create user accounts for teachers and students
5. **Start teaching digitally with INR99!**
