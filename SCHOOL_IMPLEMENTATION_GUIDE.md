## 🚀 Quick Start for Schools

### Step 1: Installation (30 minutes)
```bash
# Download and install Offline School Server
curl -fsSL https://raw.githubusercontent.com/Crosstalk-Solutions/project-nomad/refs/heads/main/install/install_nomad.sh -o install_school_server.sh
sudo bash install_school_server.sh
```

### Step 2: Initial Setup (15 minutes)
1. Open browser to `http://localhost:8080` or `http://YOUR_SERVER_IP:8080`
2. Run "School Setup" wizard
3. Select your school type (Primary/Secondary)
4. Download appropriate content collections
5. Create administrator account

### Step 3: User Management (20 minutes)
1. Create teacher accounts
2. Set up student classes and sections
3. Generate student login credentials
4. Distribute login information to staff and students

### Step 4: Start Teaching!
Your school is now ready for digital learning without internet dependency.
